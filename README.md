项目包含：
	1.本学期班级活动统计表.csv
	2.抽签程序.py
	3.名单.csv
	4.class_draw_data_backup.jason
	5.class_draw_data.jason

使用说明：请将项目包解压获得“PythonProject1”文件，并整个导入 Pycharm 中运行

规则说明：
	1.现有活动分为小型活动（A 类 如：专家讲座）和大型活动（B 类 如：足球赛）。创建 A 池和 B 池
	2.每次活动进行抽签。自愿参加的和抽签选中的同学会从对应的池子中移除。在池子重置前不会被抽中
	3.池子重置条件：
		1.一学年结束
		2.池子已被抽空
